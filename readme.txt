WP Easy Leads is a lead generating plugin that lets you create your own lead page to capture lead.

== Description ==

WP Easy Leads is a lead generating plugin that lets you create your own lead page to capture lead. It allows you to upload video or images or simply add text to the page. It allows you to add Google Analytics ID and you can also add your custom html form code. It gives users multiple options to customize the page according to their needs. WP Easy Leads allows you to create multiple lead generating pages and display the one you like.

== Installation ==
Here is what you need to do:
<ol>
    <li>Upload the plugin folder to the /wp-content/plugins/ directory or install via the Add New Plugin menu.</li>
    <li>Go to your plugin settings and enable the plugin.</li>
    <li>In the left sidebar you will have a new section called 'Leads Settings' navigate there and set your preferences.</li>
    <li>In the left sidebar you will also see a section named 'WP Leads'. Here you can create as many leads as you want.</li>
</ol>


== Frequently Asked Questions ==

FAQ

<strong>Is WPLead Form free?</strong>
Yes, it is free for the general users. <br>

<strong>Can I add my custom form?</strong>
Yes, you can add your own custom form but it has to be in an HTML format. <br>
You can add your custom form in, Go into the WordPress Dashboard on the left sidebar look for Wp Leads > Edit your lead page > Leads custom fields > and your custom form code into the Form Code area.

<strong>What size is the video that will be displayed?</strong>
By default the videos are set to 100%. You don't have to do anything with the size.

<strong>Can I change the submit button?</strong>
Yes, you can. Go into the WordPress Dashboard on the left sidebar look for Wp Leads > Edit your lead page > Leads custom fields > Submit Button Class > Chose the style you like.

<strong>Can I change the Button text?</strong>
Yes, use the short code to call the buttons and change the text of it.

<strong>Can I change the action method?</strong>
Yes, you can. Go into the WordPress Dashboard on the left sidebar look for Wp Leads > Edit your lead page > Leads custom fields > Form Action > Provide the form Action.

<strong>Can I use custom CSS?</strong>
No, currently a user can't add css.

<strong>Can I add my Google Analytics ID?</strong>
Yes, you can. Go into the WordPress Dashboard on the left sidebar look for Leads Settings > Enter Your Google Analytics ID in the field.

<strong>Can I change the background?</strong>
Yes, you can. Add a featured image it will show up as a background.
